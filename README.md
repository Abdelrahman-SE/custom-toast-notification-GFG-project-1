# Custom Toast Notification System

A fully custom toast notification component built with **Vanilla JavaScript**, **HTML**, and **CSS Animations**.

This project focuses on **clean architecture**, **DOM manipulation**, and **scalable UI logic** without using any external libraries.

## Overview

The goal of this project is to demonstrate the ability to:

- Build reusable UI components from scratch
- Control UI behavior using `data-*` attributes
- Write maintainable, production-ready frontend code

## Key Features

- Four toast types: Success, Error, Info, Warning
- Smooth enter and exit animations
- Animated progress bar with configurable duration

## Technical Highlights

## 1. Data-Driven UI

- All toast behavior is controlled using `data-type` attributes, allowing easy extension without changing core logic.
